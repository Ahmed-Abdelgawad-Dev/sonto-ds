# SONTO — Digital Solutions for Tomorrow

A modern landing page built with Vue 3, Vite, and Tailwind CSS.

## 🚀 Deployment

This project is deployed to **GitHub Pages** with automatic CI/CD via GitHub Actions.

### How it works

1. Push changes to the `main` branch
2. GitHub Actions automatically runs the workflow in `.github/workflows/deploy.yml`
3. The project is built and deployed to GitHub Pages

### Live Site

👉 [https://ahmed-abdelgawad-dev.github.io/sonto-ds/](https://ahmed-abdelgawad-dev.github.io/sonto-ds/)

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
